## 🎨 设计理念

ClipForge 是一款**效率工具**，目标用户是每天需要频繁调用剪贴板的开发者和设计师。界面设计的核心原则：

- **极简而不简陋** — 去掉所有非必要装饰，只保留让任务更快完成的元素
- **信息密度适中** — 每条记录一眼就能看清内容类型和预览，不需要滚动
- **可访问优先** — 对比度全部符合 WCAG AA（正文 ≥ 4.5:1，大文字 ≥ 3:1）
- **暗色/浅色均一流** — 两种主题完全对等，不是互相的降级版本

## 🎨 色彩系统

### 品牌色 — Teal-Slate

使用 **OKLCH 色彩空间**（感知均匀，比 HSL 更准确）。品牌色选择蓝绿 185° hue，对应"精确、高效、可信赖"的产品调性。

```css
--brand-500: oklch(54% 0.130 185);  /* 主操作色 */
--brand-400: oklch(65% 0.120 185);  /* 悬停 / 边框 */
--brand-600: oklch(44% 0.120 185);  /* 按下状态 */
```

### 中性色（含品牌色调）

纯灰色没有个性。所有中性色都加入了极微弱的品牌色调（chroma 0.005~0.010），产生潜意识上的品牌凝聚感：

```css
--neutral-50:  oklch(98%  0.005 185);  /* 页面背景 */
--neutral-100: oklch(96%  0.007 185);  /* 分区背景 */
--neutral-900: oklch(14%  0.005 185);  /* 主文字 */
```

### 内容类型语义色

每种内容类型有专属颜色，帮助用户一眼识别内容：

| 类型 | 颜色 | 值 |
|------|------|-----|
| 链接 URL | 蓝色 info | `oklch(58% 0.16 240)` |
| 邮箱 Email | 绿色 success | `oklch(58% 0.14 155)` |
| 代码 Code | 紫色 | `oklch(54% 0.20 300)` |
| 颜色 Color | 琥珀色 | `oklch(72% 0.18 65)` |
| 图片 Image | 粉色 | `oklch(62% 0.22 350)` |

### 暗色主题

暗色模式**不是简单反转**，而是重新设计：

```css
/* 表面层级通过亮度区分，不用阴影 */
--surface-base:    oklch(13% 0.008 185);  /* 最低层 */
--surface-raised:  oklch(18% 0.010 185);  /* 卡片 */
--surface-overlay: oklch(22% 0.012 185);  /* 弹窗 */
```

## 🃏 剪贴板卡片设计

### 交互状态

| 状态 | 样式 |
|------|------|
| 默认 | 白色背景，1px subtle 边框 |
| 悬停 | 品牌色边框 + shadow-md + translateY(-1px) + 左侧 3px accent 条 |
| 按下 | translateY(0) 归位 |
| 置顶/收藏 | 琥珀色边框 + 琥珀色背景淡染 |

### 左侧 Accent 条

悬停时显示 3px 宽的彩色竖条（仅左侧），颜色跟随内容类型：

- 默认文本 → 品牌色（teal）
- URL → info 蓝
- 代码 → 紫色
- 邮箱 → 绿色
- 颜色 → 琥珀色
- 图片 → 粉色

### 操作按钮可见性

**仅在悬停时显示**操作按钮（分组、删除），避免视觉干扰。

## ⚙️ 设置页设计

### 分组卡片结构

每个设置分类使用独立的 `settings-group` 卡片，卡片包含：
1. **Header** (浅色背景) — 图标 + 标题，带底部分隔线
2. **Rows** — 每行包含标签区 + 控件区，高度 ≥ 44px（触控友好）
3. 最后一行不显示底部分隔线

### 分组图标颜色

| 分类 | 颜色 |
|------|------|
| 启动行为 | Teal（品牌色） |
| 快捷键 | Amber（琥珀） |
| 数据存储 | Purple（紫色） |
| 历史记录 | Info（蓝色） |
| 自定义分组 | Success（绿色） |

## 💬 交互细节

### 过渡动效

所有过渡使用**指数缓动**（ease-out-expo），避免弹性/回弹效果：

```css
--ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);
--duration-fast: 120ms;   /* 悬停、颜色变化 */
--duration-normal: 220ms; /* 对话框、toast */
--duration-slow: 360ms;   /* 页面切换（如有） */
```

### Toast 通知

- 居中底部弹出（不是角落），避免遮挡内容
- 使用深色胶囊样式（不是信封式），紧凑无干扰
- 成功/错误用语义色区分
- 动画：`translateY(8px) → 0` + `opacity 0 → 1`

### 确认对话框

- 毛玻璃背景遮罩（`backdrop-filter: blur(2px)`）
- 对话框内含：图标（语义色背景）+ 标题 + 说明文字 + 操作按钮
- 弹出动画：`scale(0.94) translateY(8px) → scale(1) translateY(0)`

|------|---------|

## 📦 实现指南

### 需要更改的文件

1. **`src/App.css`** — 完整替换为新设计系统（参考 `docs/ui-design-preview.html` 中的 CSS）
2. **`src/App.tsx`** — 少量 JSX 结构调整（标题栏增加 app-icon，status-bar 区域）

### 关键变更点

1. **标题栏** — 增加应用图标 SVG + 条目数量徽章
2. **过滤 Tab** — 改为胶囊形，`border-radius: 9999px`
3. **剪贴板卡片** — 增加左侧 accent 条伪元素
4. **设置页** — 改为分组卡片结构（settings-group）
5. **Toast** — 改为居中底部胶囊样式
6. **状态栏** — 新增底部 28px 状态栏（条目计数 + 快捷键提示）
